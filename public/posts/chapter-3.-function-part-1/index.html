Write a function friendly(a, b) that takes two positive integers as input and returns ‘YES’ if a and b are friendly numbers. Otherwise, the function returns ‘NO’.

Note that a and b are called friendly numbers if a equals the sum of all b’s true divisors and vice versus.

For example, 220 and 284 are friendly numbers. 1+2+4+5+10+11+20+22+44+55+110=284

1+2+4+71+142=220

For example:

Test Result
print(friendly(4,5)) NO
print(friendly(220,284)) YES
print(friendly(40,50)) NO 
def friendly(a,b):
    sum_a,sum_b = [],[]
    for i in range(1,a//2+1):
        if a % i == 0:
            sum_a.append(i)
    for i in range(1,b//2+1):
        if b % i == 0:
            sum_b.append(i)
    if sum(sum_a) == b and sum(sum_b) == a:
        return "YES"
    else:
        return "NO"

Question 2: Link to heading

Write a function digit_sum_greater(a, b) that takes two positive integers a, b as input and returns ‘YES’ if the sum of all a’s digits is strictly greater than the sum of all b’s digits. Otherwise, the function returns ‘NO’

For example:

Test Result
print(digit_sum_greater(1020, 23)) NO
print(digit_sum_greater(50, 111)) YES 
def digit_sum_greater(a,b):
    sum_a = [int(num) for num in str(a)]
    sum_b = [int(num) for num in str(b)]
    if sum(sum_a) > sum(sum_b):
        return "YES"
    else:
        return "NO"

Write a function reverse( n ) that takes a positive integer as input are print n in the reverse order.

For example:

Test Result

def reverse(n):
    lst = []
    for i in str(n):
        lst.append(i)
    print("".join(lst[::-1]))

Question 4: Link to heading

Consider the following sequence:

a0=0, a1=1, an=2an-1 + an-2

Write a function calculate( n ) that takes a non-negative integer n as input and outputs the value of an.

For example:

Test Result
print(calculate(0)) 0
print(calculate(3)) 5 
def calculate(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return 2*calculate(n-1) + calculate(n-2)

For example:

Test Result

def floyd(n):
    pattern = ""
    for i in range(n):
        if i % 2 == 0:
            pattern = str(1) + pattern
            print(pattern)
        else:
            pattern = str(0) + pattern
            print(pattern)
