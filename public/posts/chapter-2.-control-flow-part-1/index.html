Write a program to count the number of even values from a to b, inclusively.

Input:

The first line contains an integer a.

The second line contains an integer b.

Guarantee that a < b.

Output:

Write out the number of even values from a to b.

For example:

Input Output
2 4
9 
a,b,cnt = int(input()), int(input()),0
for num in range(a,b+1):
    if num % 2 == 0:cnt+=1
print(cnt)

Question 4: Link to heading

Write a program to input basic salary of an employee and calculate gross salary according to given conditions.

  • Basic Salary <= 10000 : HRA = 20%, DA = 80%

  • Basic Salary is between 10001 to 20000 : HRA = 25%, DA = 90%

  • Basic Salary >= 20001 : HRA = 30%, DA = 95%

Gross salary is the final salary computed after the additions of DA, HRA and other allowances. The formula for DA and HRA is

da = basic_salary * (DA/100).

hra = basic_salary * (HRA/100).

The final gross salary is calculated as gross_salary = basic_salary + da + hra

Input:

A single line contains an positive integer which is the basic salary.

Output:

Print out the gross salary. The result should be rounded to two digits after the comma.

For example:

Input Result
500 1000.00 
basic_sal = int(input())
if (basic_sal <=10000): print("{0:.2f}".format(2*basic_sal))
elif (10001 <= basic_sal <= 20000):print("{0:.2f}".format(basic_sal + 0.25*basic_sal + 0.9*basic_sal))
else:print("{0:.2f}".format(basic_sal + 0.3*basic_sal + 0.95*basic_sal))

1000.00

Car: \$0.70/hr for the first 3 hours; $2.50/hr after 3 hours

Bus: \$1.50/hr for the first 3 hours; $2.00/hr after 3 hours

Truck: \$2.50/hr for the first 2 hour; $3.25/hr after 2 hour

Input:

The first line contains an positive integer indicating the type of a vehicle.

The second line contains an positive integer which is the number of hours spent in the car park.

Output:

Print out the total parking fee. The result should be rounded to two digits after the comma.

For example:

Input Result
2 6.50
4 
types,hours = [int(input()) for i in range(2)]
if (types == 1):
    if hours <= 3:

elif (types == 2):
    if hours <= 3:
        print("{0:.2f}".format(hours*1.5))

        print("{0:.2f}".format(3*1.5+(hours-3)*2))
else:
    if hours <= 2:
        print("{0:.2f}".format(hours*2.5))
