Question 1: Link to heading

Write a recursive function square_sum(n) to calculate the sum of first n positive square numbers: $1^2 + 2^2 +… + n^2$

Loops and iterations are NOT allowed!

For example:

Test Result
print(square_sum(3)) 14
print(square_sum(5)) 55 
def square_sum(n):
    if n == 1:
        return 1
    else:
        return n**2 + square_sum(n-1)

Question 2: Link to heading

Write a recursive function display_pascal_triangle(n) to display a Pascal triangle with n rows, where n is a given positive integer.

The $i^{th}$ row contains n-i spaces followed by Binomial coefficients C(0,i-1), C(1,i-1),…, C(i-1,i-1), which are separated by spaces, i = 1, 2,…, n

For example:

Test Result
display_pascal_triangle(6) 1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
1 5 10 10 5 1 
def display_pascal_triangle(n):
    for i in range(n):
        print((n-i-1)*" ", end = "")
        for j in range(i+1):
            print(combination(j,i),end = " ")
        print()
def factorial(num):
    if num <= 1:
        return 1
    else:
        return num*factorial(num-1)
def combination(k,n):
    return factorial(n)//(factorial(n-k)*factorial(k))

Question 3: Link to heading

The Tower of Hanoi is a mathematical problem that consists of three rods and multiple disks. Initially, all the disks are placed on one rod, one over the other in ascending order of size similar to a cone-shaped tower.

The objective of this problem is to move the stack of disks from the initial rod to another rod, following these rules:

  • A disk cannot be placed on top of a smaller disk
  • No disk can be placed on top of the smaller disk.

The goal is to move all the disks from the leftmost rod to the rightmost rod.

Your task is to write a recursive function hanoi_tower( n ) to calculate the minimum moves that need to be performed to achieve the goal.

For example:

Test Result
print(hanoi_tower(1)) 1
print(hanoi_tower(3)) 7 
#Fucking this
def hanoi_tower(n):
    return 2**n-1

#Tradition method: 
count = 0
def hanoi_tower(n):
    global count
    if n == 0:
        return
    hanoi_tower(n-1)
    count+=1
    hanoi_tower(n-1)
    return count
