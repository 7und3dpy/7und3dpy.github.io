Write a string_utilities class that contains two methods:

  • is_valid_parenthese: to validate a string of parentheses, ‘(’, ‘)’, ‘{’, ‘}’, ‘[’ and ‘]. These brackets must be close in the correct order. For example “()” and “()[]{}” are valid but “[)”, “({[)]” and “{{{” are invalid.
  • reverse_words: to reverse a string word by word.

For example:

Test Result
print(string_utilities().is_valid_parenthese(’{[(])]}’))
print(string_utilities().is_valid_parenthese(’{[()(({}))]}’))
print(string_utilities().reverse_words(‘Bach khoa Ha Noi’)) False
True
Noi Ha khoa Bach 
class string_utilities:
	def is_valid_parenthese(self, pattern):
		stack, _b, _e = "", { '(', '[', '{' }, { ')':'(', ']':'[', '}':'{' }
		for c in pattern:
			if (c in _b):
				stack += c
				continue
			if (c in _e):
				if (stack[-1] != _e[c]): return False
				stack = stack[:-1]
		if (len(stack) > 0): return False
		return True
	def reverse_words(self,pattern):
	    return ' '.join(pattern.split(' ')[::-1])

Implement the Matrix class. It should contain:

  • Constructor from a list of lists. It is guaranteed that the lists are composed of numbers, are not empty, and that they are all the same size. The constructor must copy the contents of the list of lists, i.e., when changing the lists from which the matrix was constructed, the contents of the matrix must not change.
  • The str method, which translates a matrix into a string. In this case, elements within one line should be separated by tabs, and lines should be separated by line breaks. There should be no tab character after each line, and there should not be a line break at the end.
  • The size method without arguments, returning a tuple of the form (number of rows, number of columns).

For example:

class Matrix:
    def __init__(self,matrix):
        self.m = matrix
    def __str__(self):
        res = []
        for line in self.m:
            temp = []
            for num in line:
                temp.append(str(num))
            res.append('\t'.join(temp))
        return "\n".join(res)
    def size(self):
        return (len(self.m),len(self.m[0]))

Question 8: Link to heading

Add the following methods to the previous class Matrix:

  • add: takes a second matrix of the same size and returns the sum of the matrices.
  • mul: takes an int or float and returns a matrix multiplied by a scalar.
  • rmul: does the same thing as mul. This method will be called if the argument is on the right. To implement this method in the class code, it is enough to write rmul = mul.

Explanation:

In the following case, mul will be called: Matrix ([[0, 1], [1, 0]) * 10.

In the following case, rmul is called (since int has no mul defined for the matrix on the right): 10 * Matrix ([[0, 1], [1, 0]).

Of course, these methods should not change the contents of the matrix.

For example:

class Matrix:
    def __init__(self,matrix):
        self.m = matrix
    def __str__(self):
        res = []
        for line in self.m:
            temp = []
            for num in line:
                temp.append(str(num))
            res.append('\t'.join(temp))
        return "\n".join(res)
    def size(self):
        return (len(self.m),len(self.m[0]))
    def __add__(self,m2):
        res = []
        for row in range(len(self.m)):
            sum_row = []
            for col in range(len(self.m[0])):
                val = self.m[row][col] + m2.m[row][col]
                sum_row.append(val)
            res.append(sum_row)
        return Matrix(res)
    def __mul__(self,alpha):
        res = []
        for row in range(len(self.m)):
            mul_row = []
            for col in range(len(self.m[0])):
                val = self.m[row][col] * alpha
                mul_row.append(val)
            res.append(mul_row)
        return Matrix(res)
    __rmul__ = __mul__

Let an instance M of the Matrix class is a square matrix that defines a system of linear algebraic equations: Mx = b

Your task is to add a solve method to the previous class Matrix that accepts a row vector b of free members and returns a string consisting of float numbers that represent the solution of the system of linear equations. The float numbers are separated by spaces and each number is displayed using two digits after the decimal points.

It is guaranteed that the system has always a unique solution.

For example:

class Matrix:
    def __init__(self,matrix):
        self.m = matrix
    def __str__(self):
        res = []
        for line in self.m:
            temp = []
            for num in line:
                temp.append(str(num))
            res.append('\t'.join(temp))
        return "\n".join(res)
    def size(self):
        return (len(self.m),len(self.m[0]))
    def transpose(self,matrix):
        transpose = [[0 for _ in range(len(matrix[0]))] for _ in range(len(matrix))]
        for i in range(len(matrix)):
            for j in range(len(matrix[0])):
                transpose[j][i] = matrix[i][j]
        return transpose
    def minor(self,matrix, row, col):
        return [row[:col] + row[col+1:] for row in (matrix[:row]+matrix[row+1:])]
    def det(self,matrix):
        n = len(matrix)
        if n == 1:
            return matrix[0][0]
        if n == 2:
            return matrix[0][0] * matrix[1][1] - matrix[0][1] * matrix[1][0]
        d = 0
        for col in range(len(matrix[0])):
            d += ((-1) ** col) * matrix[0][col] * self.det(self.minor(matrix, 0, col))
        return d
    def solve(self,m2):
        res = []
        if self.det(self.m) != 0:
            n = len(self.m)
            for i in range(n):
                temp = self.transpose(self.m)
                temp[i] = m2
                res.append("{0:.2f}".format(self.det(self.transpose(temp)) / self.det(self.m)))
        return " ".join(res)

m = Matrix([[1, 1, 1], [0, 2, 0], [0, 0, 4]])
print(m.solve([2,3,5]))
