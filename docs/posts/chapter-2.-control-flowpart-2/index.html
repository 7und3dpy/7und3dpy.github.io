Question 5: Link to heading

Write a program to check whether a number can be expressed as the sum of two prime numbers.

Input:

The first line contains an integer n which is the number of values to be checked.

Each line in the next n lines contains a positive integer number to be checked.

Output:

There are n lines. Print out in each line a word ‘YES’ if the corresponding input number can be expressed as the sum of two primes. Otherwise, print ‘NO’.

Explanation:

In the example below, 20 can be expressed as 17 + 3, while 11 can not be expressed as the sum of two primes.

For example:

Input Result
2 YES
20 NO
11 
n = int(input())
def prime(numb):
    for i in range(2,numb):
        if numb % i == 0:
            return False
    return True
for i in range(n):
    num = int(input())
    check = 0
    if num <= 3:
        print("NO")
    else:
        for j in range(1,num):
            if prime(j) == True and prime(num-j) == True:
                print("YES")
                check += 1
                break
        if check == 0:
            print("NO")
